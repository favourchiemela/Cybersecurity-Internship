# SIWES Internship - DataFingers (2025)

 I'm Favour, a Cybersecurity student at Miva Open University with a strong interest in Digital Forensics and Cyberpsychology.

This repository contains selected documents from my Student Industrial Work 
Experience Scheme (SIWES) internship at DataFingers, a Nigerian cybersecurity 
company.

## Documents

Cybercrime Psychology Analysis — A psychological breakdown of cybercrime 
  behavior, patterns, and motivations.

Demystifying the Mystery Behind Data Handling — An exploration of data 
  handling practices, privacy, and security considerations.

Investigating AI — A research piece examining AI in the context of 
  cybersecurity.

Revised Network Traffic Analysis Report — A network traffic analysis 
  report using Wireshark to identify patterns and anomalies.

## About Me

I'm passionate about the intersection of cybersecurity and human behavior, specifically how psychology connects to cybercrime.
